# 사용자 시나리오 (User Scenario) - 마이링크 (MyLink)

이 문서는 '마이링크' 사용자의 주요 이용 흐름을 시나리오 형식으로 정의합니다. 모든 시나리오는 "**사용자는 [목적]하기 위해 [행동]한다**"의 형식을 따릅니다.

---

## 1. 방문자 시나리오 (Visitor Scenario)

**방문자가 특정 소유자의 마이링크 페이지에 접속하고 링크를 이용하는 과정입니다.**

1.  **사용자는** 특정 크리에이터나 지인의 통합 링크를 확인하기 위해 **소유자의 고유 URL(`도메인.com/display_name`)에 접속한다.**
2.  **사용자는** 소유자가 누구인지 식별하기 위해 **상단의 네오브루탈리즘 스타일로 디자인된 프로필 이름(display_name), 실제 이름(username), 그리고 자기소개(bio)를 확인한다.**
3.  **사용자는** 소유자가 공유한 외부 채널로 이동하기 위해 **목록에 나열된 링크 버튼들을 탐색한다.**
4.  **사용자는** 해당 서비스의 신뢰도를 확인하기 위해 **링크 제목 옆에 자동으로 표시되는 파비콘(Favicon) 아이콘을 참고한다.**
5.  **사용자는** 최종 목적지로 이동하기 위해 **원하는 링크 버튼을 클릭하여 새 탭으로 연결된다.**

---

## 2. 소유자 시나리오 (Owner Scenario)

**소유자가 본인의 마이링크 페이지를 설정하고 관리하는 과정입니다.**

### 2-1. 가입 및 초기 설정
1.  **사용자는** 마이링크 서비스를 시작하기 위해 **Supabase Auth를 통한 구글 소셜 로그인을 진행한다.**
2.  **사용자는** 본인만의 고유 주소를 확보하기 위해 **Gmail 아이디 앞부분으로 자동 설정된 `display_name`을 확인하거나 가이드에 따라 수정한다.**
3.  **사용자는** 방문자에게 신뢰감을 주기 위해 **실제 이름(username)과 간단한 소개글(bio)을 입력한다.**

### 2-2. 링크 관리 (추가/수정/삭제)
1.  **사용자는** 새로운 채널을 홍보하기 위해 **대시보드에서 '링크 추가' 버튼을 눌러 타이틀과 목적지 URL을 입력한다.**
2.  **사용자는** 오타를 수정하거나 내용을 갱신하기 위해 **이미 등록된 링크 아이템을 직접 클릭하여 인라인 편집(Inline Edit) 방식으로 정보를 즉시 수정한다.**
3.  **사용자는** 더 이상 사용하지 않는 링크를 제거하기 위해 **삭제 버튼을 눌러 목록에서 해당 항목을 삭제한다.**
4.  **사용자는** 변경된 내용이 방문자에게 어떻게 보일지 확인하기 위해 **오른쪽에 실시간으로 렌더링되는 미리보기(Preview) 화면을 체크한다.**
